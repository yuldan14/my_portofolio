Poverty Level Classification Analysis in Indonesia

In this project, I developed an analysis to predict the poverty rate in Indonesia. Based on the dataset obtained from Kaggle, I identified several variables that can predict the poverty rate in Indonesia. Some of the variables used for prediction are:

  1. Province
  2. District/City
  3. Percentage of Poor Population
  4. Average Years of Schooling
  5. etc

This dataset consists of 999 rows of data, but after analysis, 485 rows were found to contain null values, so they need to be handled by removing those rows. Additionally, the issue of imbalanced data needs to be addressed. In this case, I am using the SMOTE model, which can handle data imbalance.

Thanks to this project, I was recognized as the top participant in the Talentclass Data Science Fundamental Class organized by Talenthub and the Ministry of Manpower of the Republic of Indonesia.
